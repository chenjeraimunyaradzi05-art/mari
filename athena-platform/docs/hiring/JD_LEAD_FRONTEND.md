# JOB DESCRIPTION: Lead Frontend Engineer

**Role:** Lead Frontend Engineer  
**Location:** Sydney, Australia (Remote Friendly)  
**Equity:** 0.5% - 1.5%  
**Salary:** AU$130k - $150k + Super

## The Role
You will architect the client-side experience of ATHENA. You will build a responsive, accessible, and performant web application that serves as the daily operating system for our users.

## Responsibilities
- **Development:** Build the core web application using Next.js (React) and TypeScript.
- **UI System:** Implement a design system using TailwindCSS to ensure consistency across the platform.
- **Performance:** Optimize Core Web Vitals, SEO, and load times.
- **Accessibility:** Ensure the platform meets WCAG 2.1 AA standards.
- **State Management:** Architect complex state using Zustand or React Context.

## Requirements
- **Seniority:** 5+ years of frontend development experience.
- **Tech Stack:** Expert in React, Next.js, TypeScript, and HTML/CSS.
- **Quality:** Strong experience with testing (Jest, React Testing Library, Cypress).
- **Mobile:** Experience with React Native is a huge plus.

## Nice to Have
- Experience with video streaming (HLS, FFmpeg).
- Experience building social feed algorithms.
