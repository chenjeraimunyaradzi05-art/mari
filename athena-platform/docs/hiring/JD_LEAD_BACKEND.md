# JOB DESCRIPTION: Lead Backend Engineer

**Role:** Lead Backend Engineer  
**Location:** Sydney, Australia (Remote Friendly)  
**Equity:** 0.5% - 1.5%  
**Salary:** AU$150k - $170k + Super

## The Role
You will own the server-side architecture of ATHENA. You will build robust APIs, manage databases, and integrate complex third-party services (Stripe, OpenAI, Auth0).

## Responsibilities
- **API Development:** Design and implement RESTful APIs using Node.js, Express, and TypeScript.
- **Database:** Manage MySQL database schema design and migrations using Prisma.
- **Microservices:** Architect specific services (e.g., Billing, Notifications) for scalability.
- **Security:** Implement best practices for data protection (OWASP top 10).
- **Mentorship:** Mentor junior engineers and conduct code reviews.

## Requirements
- **Seniority:** 5+ years of backend development experience.
- **Tech Stack:** Expert in Node.js, TypeScript, and SQL.
- **Architecture:** Experience with microservices, serverless (Lambda), and message queues (Redis/BullMQ).
- **DevOps:** Familiarity with Docker, CI/CD pipelines, and AWS services.

## Nice to Have
- Experience with payment processing (Stripe Connect).
- Experience with search engines (Elasticsearch/OpenSearch).
