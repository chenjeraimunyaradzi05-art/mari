# GDPR Compliance Checklist

## Data Access & Portability
- [ ] Export user profile, posts, messages, and settings into JSON/ZIP.
- [ ] Include export request audit log.

## Right to be Forgotten
- [ ] Soft-delete user account, anonymize references.
- [ ] Cascade deletion for personal identifiers.

## Consent & Preferences
- [ ] Record consent timestamps for email/push/in-app.
- [ ] Provide preference center with audit trail.

## Security
- [ ] Encrypt sensitive data at rest.
- [ ] Log access to personal data.

## Operational
- [ ] Add DPA and breach notification procedure.
- [ ] Keep data retention policy updated.
